# Valorant UTOC Signature Bypass

This patch enables Valorant to load custom `.pak`, `.utoc`, and `.ucas` files without requiring `.sig` signature files.

## Overview

Valorant uses `.usmap` mappings to define the internal structure of `.uasset` files.  
Mods built with incorrect or outdated mappings will not load.
